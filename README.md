# AnovaGrowth AI Labs Research Platform

A Next.js research platform for AI breakthrough discovery.

## Development

```bash
npm install
npm run dev
```

## Deployment

This site deploys to `research.anovagrowth.com`

```bash
npm run build
vercel --prod
```

## Content Structure

- `/content/research/` — Published research articles
- `/content/theories/` — Theoretical frameworks
- `/content/lab-notes/` — Raw research notes
- `/content/breakthroughs/` — Major advances

## Research Schedule

- Every 6 hours: Deep research cycle
- Every 12 hours: Publish synthesis article
- Weekly: Collaborative review
